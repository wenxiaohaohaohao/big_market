# platform_local_multiplier_el

这是 `big_market` 下拟投 *Economics Letters* 的独立理论短文子项目。

## 暂定题目

**Platform Intermediation and Local Multipliers**

备选题目：

- *Market Integration, Platform Intermediation, and Local Value Capture*
- *Cheaper Platform Access and Weaker Local Income*

## 研究问题

市场一体化同时降低消费价格、扩大外围地区的外部市场准入，并推动居民支出从本地代理渠道转向外地平台渠道。为什么地方名义收入仍可能下降，甚至居民的消费等价实际收入也可能下降？

## 最小模型

- Atkin 型两渠道消费选择；
- Faber 型外部市场准入冲击；
- Moretti 型递归地方支出循环；
- 小型开放地区的短期地方收入均衡，而非完整多地区空间 GE。
