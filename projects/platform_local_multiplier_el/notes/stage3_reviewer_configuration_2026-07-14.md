## 2. 期刊匹配

1. **Economics Letters（首选）**：其 scope 接受经济学各领域的新结果、
   模型与方法，允许理论作者用 concise letter 先报告
   thought-provoking analytical result；通常 2,000 词的限制与当前稿件
   直接匹配。Stage 3 必须严格检查稿件是否超过 pedagogical note 或
   obvious example 的边界。
2. **Economic Theory Bulletin（理论压力测试）**：形式上接受 rigorous
   short theoretical papers，但明确排除已知结果的简单推论。它不是当前
   同等推荐目标，而是检验 novelty 是否足够实质性的更高门槛参照。
3. **Papers in Regional Science（主题匹配、格式不匹配）**：地方收入、
   空间归属和区域乘数符合 scope，但通常需要比现有 letter 更完整的空间
   结构、机制或应用；如转投需要扩展正文，而不是只改格式。

## 3. Reviewer Configuration Cards

### Card 1：Editor-in-Chief

**Identity**  
一位负责 *Economics Letters* microeconomics、industrial organization
与 regional economics 投稿的 senior editor，长期判断 concise theory
稿件是否提供独立的新结果，而不是 pedagogical note 或已知结果的重写。

**独立审查范围**

1. 能否用一句话说明一个非显然、可独立复述且适合 general economics
   readership 的结果；
2. title、abstract、Introduction、formal results 与 conclusion 是否承诺
   同一贡献；
3. 当前 1,884 词是否全部服务于核心结果；
4. 结果是否值得以 letter 独立发表，还是只有放入完整空间模型才有意义。

**特别关注**  
编辑在 desk review 阶段能否立即看出“为什么经济学家需要知道这个结果”，
以及文章是否超越换符号后的 standard multiplier exercise。

**可能盲点**  
不逐式复核证明，也不承担穷尽相邻文献的任务。

### Card 2：Reviewer 1 — Analytical Methodology

**Identity**  
一位研究 CES expenditure systems、fixed-point equilibrium、
comparative statics 与 analytical multiplier models 的 microeconomic
theorist，专门审查短理论稿的 existence、uniqueness、interiority、
threshold exhaustiveness 和 limiting arguments。

**独立审查范围**

1. 从 primitives、household problem、channel expenditure minimization
   到 local-income fixed point 是否闭合；
2. \(m_B\) 与 \(\mathcal M_G\) 的偏导对象、固定条件和经济解释；
3. Proposition 2 三个区域、等号边界与不存在第四种情形的完备性；
4. \(s^\dagger<1/2\) 的唯一性、可达性、global maximum 与 \(G=0\)
   限定；
5. corners、\(\eta\to1,\infty\)、\(\rho_A\leq\rho_P\) 与 \(G>0\)
   边界的一致性。

**特别关注**  
结果是否真正依赖经济结构，而不是仅对预先构造的 closed form 做微积分。

**可能盲点**  
不负责文献 novelty，也不重点评价平台制度含义。

### Card 3：Reviewer 2 — Domain and Literature

**Identity**  
一位研究 local multipliers、place-based shocks、地方所有权、市场整合
与区域价值捕获的 regional and urban economist，熟悉 Moretti、
Chodorow-Reich、Faber、Atkin、Couture、Fan、Goldmanis、Brown 及近期
e-commerce spatial models。

**独立审查范围**

1. local-income retention 与 geographic multiplier、regional
   input-output leakage、local ownership 文献的关系；
2. 相对 Atkin、Fan、Couture、Li，本文新增的 mechanism/result；
3. recurring leakage、price--income divergence、channel substitution
   是否充分让步；
4. \(B(z)\)、\(G\)、\(\rho_c\)、nominal income 与
   consumption-equivalent real income 的术语是否规范；
5. 是否遗漏直接相邻的 analytical local-retention model。

**特别关注**  
贡献是否具有 field-specific substance，而不是给已知地方乘数加上
channel-weighted leakage rate。

**可能盲点**  
不逐式复证明，也不要求完整平台合同模型。

### Card 4：Reviewer 3 — Cross-disciplinary Spatial Incidence

**Identity**  
一位研究 retail IO、digital intermediation、supply-chain incidence
与 spatial ownership 的 applied micro theorist，关注价格、佣金、履约、
企业所有权和交易地点如何决定消费支出的空间归属。

**独立审查范围**

1. 用外生 \(\rho_A,\rho_P\) 压缩渠道差异后是否仍有清楚的 economic
   incidence content；
2. \(p_P(z)\) 下降可对应哪些机制，不同解释是否改变收入归属；
3. “platform channel retains less”作为条件命题是否兼容本地卖家、配送、
   仓储和所有权安排；
4. 模型是否给出可观察、可反驳的 prediction；
5. 机制迁移到 national chains、remote services 或外地所有渠道时，
   是否强化或削弱平台动机。

**特别关注**  
\(\rho_c\) 是否有足够具体的经济解释，使稿件成为 spatial incidence
theory，而不是任意两个抽象渠道的代数例子。

**可能盲点**  
可能高估制度微观基础要求，需要 EIC 按 letter scope 校准。

### Card 5：Devil’s Advocate

**Identity**  
一位对 minimal analytical models 高度怀疑的 theorist/editor，专门识别
accounting identity presented as theorem、known multiplier relabeling
以及由 functional form 机械产生的 novelty claims。

**独立审查范围**

1. 尝试把模型改写为标准两部门 multiplier，检验 retention-feedback
   是否只是 leakage parameter 的重新命名；
2. 检验外生 \(\rho_c\) 与维护的 \(B(z)\) 是否已把 integration 的实质
   作用预先写入假设；
3. 检验 \(s^\dagger<1/2\) 是否只是 CES logistic share 与线性 retention
   的函数形式产物；
4. 换成一般 channel-share function 或 nonlinear retention 后，哪些
   threshold 与 peak 仍成立；
5. 进行“so what?”测试；
6. 寻找 platform theory、market integration theory 或 welfare theory
   的 scope overreach。

**特别关注**  
为什么文章不是“假定平台留值较低，所以地方收入较低”的循环论证，以及
闭式阈值是否提供了超越方向性结论的信息。

**可能盲点**  
可能系统性低估一个有意最小化的 analytical letter 的价值；EIC 需区分
致命反例与可在 scope statement 中承认的扩展。
