# NSE 过渡性论文项目备忘录

日期：2026-07-24

状态：活动项目主轴（`CONFIRMED`）

## 一、论文身份

本文属于“统一大市场、平台化与地方发展”研究计划，是在 EL 短文基础上向生产侧
推进的独立理论论文。

“独立”是指新稿必须建立企业进入、比较优势实现、组织选择和支付归属的微观基础，
不能把 EL 中的外生本地能力简单改写为 NSE 术语；“在 EL 基础上推进”是指新稿
继续解释同一个核心问题：

> 为什么平台化市场整合能够普遍改善消费者接入，却只有在潜在比较优势能够通过
> 适宜的本地组织和基础设施条件得到实现时，才会转化为本地生产和地方收入？

因此，本文不是一般性的 LCA--ACA 论文，也不是一篇完整 NSE 论文附带平台应用。
平台化造成的 `market access--local capture` 分离是研究对象；NSE 为地方发展侧
提供结构骨架，贯穿禀赋与产业选择、LCA--ACA、viability、组织嵌入、本地产业
联系、capture formation 和 facilitating-state 边界。

## 二、不可改变的主从关系

```text
研究对象：
平台化市场整合
  -> consumer-side access
  -> producer-side access
  -> local value capture
  -> nominal and consumption-equivalent local income

解释机制：
禀赋结构
  -> 相对生产成本
  -> latent comparative advantage
  -> transaction and organizational costs
  -> actual comparative advantage
  -> viability and realized firm entry
  -> local organizational embeddedness
  -> local complementary services and linkages
  -> endogenous local production and capture

条件性政策：
facilitating state
  -> 降低潜在比较优势产业的共享性真实交易成本
```

NSE 模块可以解释上述地方发展转化链的多个环节，但每个环节都必须回到
access--capture 结果。不能因为某一 NSE 模块本身可建模，就将其提升为全文新的
研究问题。消费者偏好、平台技术冲击和 EL 收入乘数仍使用各自理论来源，不被强行
改写为 NSE。

## 三、与 EL 的关系

EL 短文在给定渠道留值和地方结构时，分解平台化的消费者接入收益、本地捕获损失
及收入传播。

本稿内生化两个对象：

1. 本地生产者是否进入以及销售规模怎样响应统一市场；
2. 生产和交易所需服务采用何种组织方式，支付最终归属于哪里。

均衡结果为：

```text
endowments and infrastructure
  -> firm entry and organizational choice
  -> first-round local income and retention
  -> EL-consistent local-income propagation
```

新稿不得只写成 `kappa_i = kappa(CA_i,G_i)`。企业进入、组织方式和支付流必须来自
利润比较或资源约束。

EL 的完整固定点不进入生产侧资源约束，但 access--capture 分解必须进入正文基准；
收入传播公式可以在正文末端或附录中按 EL 原式映射。

## 四、最小模型的三个 block

### Block A：消费者接入与本地渠道替代

- 平台化降低本地居民购买外部商品的价格或交易成本；
- 平台份额上升、消费价格指数下降；
- 外部平台和外地商品替代本地代理、批发、零售或交易服务；
- 支付流决定第一轮 inbound transaction 中有多少价值留在本地。

这个 block 产生 consumer-access gain 和 capture loss。

### Block B：NSE 结构实现与本地组织嵌入

- 地区禀赋和行业要素密集度决定相对生产成本；
- 相对生产成本定义 LCA，而不是使用外生匹配指数；
- 全国市场接入和软硬基础设施决定总交易成本；
- 当 LCA 产业在现有软硬基础设施下形成相对总成本优势时，它由 LCA 转化为
  actual comparative advantage（ACA）；
- viability 表示企业在开放竞争市场中不依赖持续保护或补贴而获得社会可接受
  正常利润的能力；企业进入和生产是 ACA 得到实现的均衡表现，不是 ACA 的定义；
- 进入全国市场以后，企业在外部平台组织与本地嵌入式组织之间选择；
- 组织选择决定履约、认证、营销和供应链服务由本地还是外部主体提供；
- 本地产业联系是同一交易所需配套服务的本地化结果，不是另加一个外生留值能力。

这个 block 同时产生 producer-side gain、local-service gain 和 external-payment
incidence。NSE 因而影响 production 和 capture，而不只影响企业是否进入。

### Block C：地方收入与实际收入

第一轮地方收入变化分解为：

```text
local producer income
+ local platform-complementary service income
- displaced local-agent income
- payments to external platforms and services.
```

随后分别计算：

- nominal local income；
- consumer price index；
- consumption-equivalent local income。

全文必须能够产生“消费者接入改善但名义地方收入下降”的参数区域。

## 五、核心命题层级

1. **比较优势实现命题**：更强的 LCA 降低平台市场进入所需的基础设施条件。
2. **组织嵌入命题**：本地组织和共享基础设施决定进入后的生产及配套服务有多少
   留在当地。
3. **名义地方收入阈值**：只有 producer-side 和 local-service gains 足以抵消
   channel displacement 与 external payments 时，
   平台化才提高名义地方收入。
4. **实际收入阈值及排序**：消费价格收益使实际收入阈值严格低于名义收入阈值。
5. **facilitating-state 推论**：具有共享性市场失灵时，降低真实交易和组织成本与
   通过保护提高外部渠道成本不等价。

基准模型保留最小的“外部平台组织--本地嵌入式组织”选择，用于生成本地服务支付
和外部支付。旧稿完整的 \(M_P,M_{L0},M_X\) 三阈值体系仍只作为扩展，不再承担
全文主命题。

## 六、第一版明确排除

- 完整 New Economic Geography；
- 迁移、土地、住房和内生城市规模；
- 动态组织资本；
- 多平台战略定价和双边网络效应；
- 完整多地区一般均衡；
- 未获得数据的空实证章节；
- 以 local value capture 代替 national welfare。

## 七、无数据理论稿的完成标准

- consumer access、producer access、capture 和 local income 同时进入模型；
- LCA、ACA 和 viability 使用严格且不混淆的定义；
- 企业进入、组织嵌入和支付归属由行为方程决定；
- NSE 同时生成 local production 和 local complementary-service income；
- 至少证明名义与实际收入阈值及其严格排序；
- facilitating state 对应一个明确的公共品或协调失败；
- numerical exercises 只称 illustrative，不称 calibration；
- 所有引用进入 `references.bib` 前完成元数据核实；
- legacy 文件不再作为当前模型来源。
