# Literature Protocol

本目录只存放经过来源核实的文献材料和 manifest。聊天生成的文献清单、
搜索结果和待补条目不得直接当作 bibliography。

## 状态

| 状态 | 含义 | 可否进入 `references.bib` |
|---|---|---|
| `CANDIDATE` | 只有题名或检索线索 | 否 |
| `METADATA VERIFIED` | 作者、年份、题名、载体、卷期页码或 DOI 已由权威来源确认 | 可以 |
| `FULL TEXT CHECKED` | 已核对原文与拟引用论断 | 可以 |
| `CITED` | 已在正文使用并完成 claim-reference alignment | 可以 |

## 当前审计结论

2026-07-23 审阅时，旧材料声称“74 篇文献已整理”，但本目录没有 PDF，
`paper/references.bib` 也不存在。准确表述应是：曾形成候选文献与变量清单，
尚未完成逐条可引用性核实。

抽查发现：

- “新结构经济学视角下的平台经济：一个分析框架”存在正式来源，但旧清单
  把它记为 2025 年研究院工作论文；正式来源显示为吴双、王勇发表于
  《浙江社会科学》2024 年第 4 期，DOI `10.14167/j.zjss.2024.04.001`。
- 林毅夫“新结构经济学的理论内核与创新贡献”可由北京大学新结构经济学
  研究院页面确认发表于 2026 年第 1 期，但正式 BibTeX 仍需补全期刊信息。
- 部分中文条目缺作者、年份、期刊或卷期；部分 2026 年经验研究的样本数量
  在同一清单不同位置不一致。
- 旧清单中至少两处 Markdown 表格公式被竖线破坏。

如找回旧候选清单，应先保存为 `candidate_inventory.md`，再建立
`source_manifest.csv`，逐条核实后写入 BibTeX。
