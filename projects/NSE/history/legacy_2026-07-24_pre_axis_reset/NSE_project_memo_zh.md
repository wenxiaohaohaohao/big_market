## 二、核心理论链条

```
要素禀赋 E_i = (K_i, L_i, N_i)
    ↓ 决定要素价格 (w_i, r_i)
行业单位成本 c_{ij} = c_j(w_i, r_i; θ_j)
    ↓ θ_j 为行业要素密集度
潜在比较优势产业集（c_{ij} 最低的那些 j）
    ↓
异质企业进入：生产率 z，选择行业 j 和渠道 r ∈ {A, D}
    利润 π_{ij}^r(z) = R_{ij}^r(z; τ, φ_r, M) - VC_{ij}(z) - F_{ij}^r(q_i, G_i)
    ↓ 零利润条件 → 进入门槛 z_{ij}^{r*}
均衡本地价值捕获 ρ_i
    第一轮本地收入 I_i = W_i + Π_i^local + S_i^local + T_i^local
    ρ_i = I_i / 与相关交易对应的总价值
    ↓
地方总收入 Y_i（通过非贸易部门再循环）
```

### 关键创新点

1. **ρ_i 是均衡结果，不是外生参数**——这是与 EL 短文的根本区别
2. **消费侧 access 与生产侧 access 被明确区分**：
   - 消费侧：本地居民更容易买到外地商品
   - 生产侧：本地企业更容易把商品卖到全国市场
3. **retained value 不是社会福利充分统计量**——必须区分地方收入、地方政府目标、全国社会福利

## 三、三个核心命题

### 命题一：比较优势实现与平台进入阈值

平台化降低全国市场进入成本 τ，但其生产扩张效应在比较优势
一致型产业中更强：

∂z_{ij}^{D*} / ∂τ

取决于地方单位成本和行业要素密集度。

**经济学含义**：相同统一市场机会，在不同地方和行业产生不同进入反应。

### 命题二：组织能力阈值

存在平台兼容型组织能力阈值 q_i^*，比较优势越强，q_i^* 越低。
当 q_i ≥ q_i^* 时，平台接入带来的外部销售扩张足以补偿佣金和
传统中介收入损失。

更强的 NSE 预测：

∂q_i^* / ∂(comparative-advantage strength) < 0

### 命题三：赋能与保护的条件性比较

在存在协调失败或公共品外部性时，能力建设 G_i 可以降低新组织
能力形成成本，地方保护主要提高对本地产品的被动需求。

在潜在比较优势足够强的行业中，W^facilitation > W^protection。
但这个结论必须有条件，不能宣称为无条件 NSE 政策结论。

## 四、关于 NEG 的决定

**第一版不引入新经济地理学。**

理由：

1. NEG 一旦加入市场规模、迁移和集聚外部性，地方差异可能来自
   home-market effect，稀释 NSE 主轴
2. NEG 本身是一个完全可以独立发表的方向
3. 第一版先把 NSE 机制做扎实，向林老师汇报时主动说明 NEG 是
   后续扩展方向
